# 아기 사진 자랑하기


## '아기 사진 자랑하기'가 뭔가요?

아기가 태어날 때부터 돌 때 까지의 사랑스런 모습이 담긴 사진들을



개월 수 별로 정리해서



멋지게 사이트에서 볼 수 있도록 도와주는 프로젝트입니다.







## 실행 


첫 화면에서 제작하기 버튼을 누릅니다.


두번쨰 페이지에서


아빠 이름, 엄마 이름, 아기 이름, 아기가 태어난 일자, 시간을 적습니다.


태어난 날의 사진부터 50일, 100일, 4개월...12개월 까지의 사진을 차례대로 넣습니다.


저장하기 버튼을 누릅니다.



결과 페이지로 이동하여 


사랑스런 아기의 사진들을 감상하고 



주변 사람들에게 링크를 보내어 공유해보세요!
